# 🧪 RS PathLab – Diagnostic Appointment Booking System

RS PathLab is a web-based diagnostic appointment booking platform designed to simplify healthcare access. It allows users to book lab tests, explore health packages, and manage appointments with ease.

---

## 🚀 Features

- 🛒 Browse and select lab tests & packages  
- 📅 Book appointments with preferred date & time  
- 🏠 Home sample collection सुविधा  
- 📲 Reports delivered within 24 hours (via WhatsApp)  
- 🩻 Digital X-Ray service integration  
- 📱 Fully responsive and user-friendly UI  

---

## 🏥 About the Project

RS Path Lab is a trusted diagnostic center located in Jamshedpur, Jharkhand. This project aims to digitize the appointment booking process and improve accessibility to diagnostic services.

The platform focuses on:
- Accuracy and reliability in healthcare services  
- Easy online booking system  
- Affordable diagnostic solutions  
- Fast and convenient report delivery  

---

## 🔬 Services Included

- Complete Blood Count (CBC)  
- Digital X-Ray  
- Thyroid Profile (T3, T4, TSH)  
- Diabetes Panel  
- Cardiac & Lipid Profile  
- Full Body Health Checkup Packages  

---

## ⚙️ Tech Stack

- HTML  
- CSS (Tailwind CSS)

- React.js  

---

## 📋 How It Works

1. Choose required tests or packages  
2. Select date and time slot  
3. Book appointment online  
4. Sample collection at home  
5. Receive reports digitally
